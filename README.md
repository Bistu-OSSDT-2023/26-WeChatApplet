## 项目介绍 

 本项目为约课小程序，使用微信小程序开发平台进行开发，可以为各类培训机构提供学生家长选课、预约时间的登记，可以查看自己的课时消耗情况。学生端可以选择老师和预约时间，预约成功后课时将减少，若取消预约，将恢复课时数。老师端可以设定预约时间段及人数。后台端可以设置老师及课程的基本信息，还可给学生设置课时数。
